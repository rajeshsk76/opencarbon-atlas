# Data Sources

Source metadata lives in `data/norway/references.csv`. Each source has a stable `source_id`, publisher, URL, access date, and note.

The seed dataset references public facility and project pages from:

- Miljødirektoratet Norske utslipp
- Heidelberg Materials Norway
- Hafslund Oslo Celsio
- Equinor
- Gassco
- TiZir

Values in the sample data are intended to establish repository structure and validation behavior. Production use should verify every quantitative value against primary source material.
