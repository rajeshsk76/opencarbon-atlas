# Roadmap

## v1

- Norway-only CSV dataset.
- FastAPI read API.
- Leaflet point map.
- Validation and GeoJSON build scripts.
- Methodology and schema documentation.

## Later

- Add automated imports from authoritative public datasets.
- Add versioned snapshots and change logs.
- Add richer capture-cost and transport-distance calculations.
- Add authenticated editorial workflows.
- Expand beyond Norway after the Norway schema stabilizes.
